# Collage Mobile Pro

نسخة احترافية من أداة دمج الصور، مبنية بـ **Python + Flask + Pillow**، ومصممة لتعمل بشكل ممتاز على **الجوال** و**الكمبيوتر** عبر المتصفح.

## لماذا هذه النسخة أفضل؟

لأنك تريد مشاركة الأداة مع زملائك على الجوال، فالأفضل ليس Tkinter، بل **Web App**:

- يفتحون رابط فقط من الجوال.
- يرفعون عدة صور.
- يضيفون عنوانًا أعلى الصورة.
- يضغطون زر إنشاء.
- ينزلون الصورة النهائية مباشرة على أجهزتهم.

## الميزات

- رفع **10 صور أو أكثر**.
- دعم الصور ذات الأبعاد المختلفة.
- إنشاء **Collage** تلقائي بشكل مرتب.
- إضافة عنوان مع:
  - النص
  - حجم الخط
  - المحاذاة
- ضغط ذكي للصورة النهائية لتبقى تحت الحجم المطلوب.
- واجهة متجاوبة مناسبة للجوال.
- حفظ النتائج في مجلد `outputs`.
- تنزيل مباشر للنتيجة.
- بنية منظمة وقابلة للتطوير.
- دعم Windows / macOS / Linux.
- مناسب للنشر الداخلي داخل الشبكة أو على استضافة خارجية.

## تشغيل المشروع

### 1) إنشاء بيئة افتراضية

```bash
python -m venv .venv
```

**Windows**
```bash
.venv\Scripts\activate
```

**macOS / Linux**
```bash
source .venv/bin/activate
```

### 2) التثبيت

```bash
pip install -e .
```

### 3) التشغيل

```bash
python -m collage_web.app
```

أو:

```bash
collage-mobile-pro
```

ثم افتح المتصفح على:

```text
http://127.0.0.1:5000
```

## مشاركة الأداة مع الزملاء على نفس الشبكة

شغّلها هكذا:

```bash
python -m collage_web.app --host 0.0.0.0 --port 5000
```

ثم اعرف عنوان IP الخاص بجهازك، مثل:

```text
http://192.168.1.20:5000
```

وارسل هذا الرابط لزملائك إذا كانوا على نفس شبكة الواي فاي أو الشبكة الداخلية.

## نشرها على الإنترنت

يمكنك نشرها بسهولة على:

- Render
- Railway
- VPS Ubuntu
- Docker
- داخل سيرفر الشركة

### تشغيل Gunicorn مثال:

```bash
pip install gunicorn
gunicorn -w 2 -b 0.0.0.0:5000 "collage_web.app:create_app()"
```

## ملاحظات مهمة بخصوص الجوال

- **المتصفح لا يستطيع إجبار النظام على الحفظ داخل Gallery مباشرة**.
- لكن يمكن تنزيل الصورة، وبعدها المستخدم يحفظها في الصور/المعرض حسب نظام الجهاز.
- في بعض الجوالات، التنزيل يظهر تلقائيًا في الملفات أو الصور.

## الاستخدام

1. افتح الصفحة من الجوال أو الكمبيوتر.
2. اختر عدة صور.
3. اكتب عنوانًا إذا أردت.
4. اختر حجم الخط والمحاذاة.
5. حدد الحد الأعلى للحجم النهائي.
6. اضغط **Create Collage**.
7. نزّل الصورة الناتجة.
